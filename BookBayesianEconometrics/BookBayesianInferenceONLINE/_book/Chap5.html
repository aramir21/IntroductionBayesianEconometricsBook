Introduction to Bayesian Data Modeling

Chapter 5 Graphical user interface

This chapter presents our graphical user interface (GUI) to carry out Bayesian regression analysis in a very friendly environment without any programming skills (drag and drop). Our GUI is based on an interactive web application using shiny (Chang 2018), and packages like MCMCpack (Martin, Quinn, and Park 2018) and bayesm (Rossi 2017) from R software (R Core Team 2023), and is designed for teaching and applied purposes at an introductory level. In the next chapters of the second part of this book, we carry out some applications to highlight the potential of our GUI for applied researchers and practitioners.
